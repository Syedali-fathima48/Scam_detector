# 🔐 Scam Detector

## 📌 Description
Scam Detector is a web-based application that helps users identify whether a message is safe or a potential scam. It analyzes user input and classifies it as Safe, Suspicious, or Scam.

## 🚀 Features
- Detects scam messages
- Classifies text as Safe / Suspicious / Scam
- Simple and user-friendly interface
- Instant result display

## 🛠️ Technologies Used
- Python (Flask)
- HTML

## ▶️ How to Run

1. Run the app:
   python app.py

2. Open in browser:
   http://127.0.0.1:5000

## 💡 Future Scope
- Multi-language support
- AI-based advanced detection
- Integration with apps like SMS or email
